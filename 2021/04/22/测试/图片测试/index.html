前言 及 介绍



前言

QQ / 微信 ，作为我们日常生活中最常使用的聊天工具，已经深深地融入进了我们的生活，突然脱离这两个工具可能会十分的麻烦，甚至会有一些【返祖】，即使你身边的人可以通过手机上的 QQ / 微信联系到你，但是当你在办公或其他状况下，需要使用 电脑 / 手机 互传照片的时候，你的电脑上没有 QQ / 微信也是十分不方便的，好在，在今年年初的时候，腾讯公司发布了 QQ for Linux 版本（之前曾经也发布过，但是下架了，我今年才发现又上线了，时间不考究），相信很多 Linux 用户已经体验过这款 QQ 带给人的惊人体验了，这仿佛是一款有魔力的 QQ，除了界面很老旧，让你看到界面的同时就开始缅怀你的童年以外，它还像微信一样，仅支持扫码登录，手机下线 == 电脑下线，嗯……不得不说这有一些不 银性 ，但是办法总比困难多，接下来介绍一个方法，使用 wine 容器承载 Windows 版本的 QQ / 微信，并且运行在 Linux 系统上；



wine 介绍

wine 是一款运行在 Linux 系统上的微型容器(没记错的话应该是由国产系统Deepin的开发团队开发的)，你可以把它看成是一个小的 VmWare for Linux 版本，它的体积是十分小的，能够在占用极少的资源的情况下，在 Linux 上完成 .exe文件的运行，所以我个人认为这是一个十分不错的工具，毕竟如果不适用 wine 的话，你看到的 QQ 是如下图一这个样子的，而用了 wine 虚拟机的你，登录变成了图二的样子，界面变成了图三的样子；

不过，如果仅仅只是丑点儿那也还是可以接受的，毕竟有总比没有好，但是 微信 甚至连 Linux 版本都没有，所以更不要说丑了，因为这一系列的原因，我是十分推崇 wine 移植的；

ok，接下来进入教学阶段；

   




安装软件



安装 wine

  • 安装一下wine环境，命令如下（git clone就是从后面的地址下载的意思，你也可以直接点进地址进行下载）：

    • （国内地址）：git clone https://gitee.com/wszqkzqk/deepin-wine-for-ubuntu.git
    • （国外地址）：git clone https://github.com/wszqkzqk/deepin-wine-ubuntu.git

  • 解压缩：

    • 方法一：进入Downloads文件夹，鼠标右键选择解压缩（Extrac Here）；
    • 方法二：使用命令cd Downloads进入下载文件夹，使用命令tar -zxvf deepin-(点tab键自动补全) -C /root/Downloads/；最后的文件夹位置可以自己更换；

  • 进入刚刚解压缩的文件夹，执行如下命令：

    • 命令：./install.sh
    • 命令：./KDE-install.sh
    • 命令：./install（tab补全）.sh



安装 QQ / 微信等软件

  • 下载：

    QQ：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im/

    QQ备用链接：https://gitee.com/wszqkzqk/deepin-wine-containers-for-ubuntu/raw/master/deepin.com.qq.im_9.1.8deepin0_i386.deb

  • 安装：

    命令：sudo dpkg -i +包名

  • 若是没有报错，就安装成功了，若是报错了，就执行sudo apt --fix-broken install 或者 sudo apt -f install -y命令解决一下以来问题，并在此执行（上一步的）安装命令；
